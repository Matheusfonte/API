# API

# Consumo de API com JavaScript

## Sobre o projeto
Este projeto foi desenvolvido como atividade prática para aprender a consumir uma API pública usando JavaScript com **fetch()**.

A aplicação carrega uma lista de posts da API JSONPlaceholder, exibe os dados na tela, permite buscar posts pelo título e mostra os detalhes quando um item é clicado.

## Tecnologias utilizadas
- HTML
- CSS
- JavaScript
- Fetch API

## O que cada um faz:
- index.html → estrutura da página
- styles.css → estilos da aplicação
- script.js → lógica e consumo da API

## API utilizada
https://jsonplaceholder.typicode.com/posts

## Funcionalidades
- Carregar dados da API
- Exibir lista de posts
- Buscar posts pelo título
- Mostrar detalhes ao clicar
- Estados de loading, erro e sem resultados

## Como executar
1. Baixar ou clonar o projeto  
2. Abrir o arquivo **index.html** no navegador
